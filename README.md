## What I've built

### 🤖 Copilot Studio Agents
Three agents I use daily to run my working life.

→ [**Daily Operations Assistant**](https://github.com/claudiafranceschini/daily-ops-agent) — Morning briefing agent that surfaces priority emails, outstanding to-dos, and meeting prep so I start every day with context, not chaos.

→ [**Follow-up & Chase Tracker**](https://github.com/claudiafranceschini/followup-chase-agent) — Monitors open threads and flags anything that hasn't had a reply in 24 or 48 hours, depending on urgency. I set the rules; it handles the memory.

→ [**Senior Stakeholder Comms Agent**](https://github.com/claudiafranceschini/stakeholder-comms-agent) — Drafts communications to senior leadership and external partners using a defined tone profile, relevant context, and audience-specific framing. Cuts drafting time significantly and reduces the back-and-forth of getting tone right.

### ⚙️ Claude Workflow Skills
A set of structured prompts and workflows I've built inside Claude to automate repeatable operational tasks.

→ [**AI Tool Evaluation Framework**](https://github.com/claudiafranceschini/claude-workflow-skills) — Automates the process of evaluating new AI tools: compliance questionnaire responses, use case scoring, and investment recommendation summaries from a single prompt.

→ **Deck & Comms Generator** — Produces first drafts of strategic decks, partner briefings, and community communications from a short brief. Reduces a 2-hour task to under 20 minutes.

→ **Partner Onboarding Workflow** — Structures the full onboarding process for new tool partners: checklist generation, stakeholder comms, and rollout tracking from a templated input.

## How I work

I use AI to multiply my output, not to replace thinking. The workflows above exist because I identified a repeatable task, worked out what good output looked like, and built something that produces that output consistently — without me having to start from scratch every time.

The goal is always the same: protect time for the work that needs a human, and automate everything that doesn't.
